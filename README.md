# Введение

RU: Тестовый проект по написанию простенькой банковской системы. Создан для изучения docker, go, работа с базами данных и использованием телеграмм бота

# Структура

- 'src/': Содержит все основные файлы для бэкенда
    - 'bootstrap': Запуск приложения
    - 'controllers': Принимает запрос-ответы от http
    - 'domain': Объекты 
    - 'repository': Запросы в базу данных
    - 'services': Бизнес-логика системы
